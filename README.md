# Quality Reflections Glasswork

Commercial glazing company website featuring:
- Custom blueprint-style cursor
- Scroll-pinned hero with exploded 3D glass assembly animation
- AI-generated project photography
- Glass-morphism UI components

## Development

Static HTML site. To run locally:

```bash
python3 -m http.server 8000
# or
npx serve
```

Then open http://localhost:8000

## Structure

```
index.html          # Main site
project-*.png       # AI-generated project images
```
